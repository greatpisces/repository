selector

jQuery选择器基于元素的id、类、类型、属性、属性值等选择（已存在的css选择器、自定义选择器）

  1. css选择器
    2. 选择器 说明
    .class 选择所有class的元素
    #id 选择id匹配的元素
    * 选择所有元素
    element 选择与element匹配元素
    element1,element2 选择所有相匹配的元素
    element1 element2 1中所有的2元素
    element1>element2 所有父级是1的2元素
    element1+element2 紧接着1的2元素
    [attribute] 所有带目标属性的元素
    [attribute=value] 所有属性匹配的元素
    [attribute~=value] 所有包含value单词的属性（模糊查找）
    [attribute|=value] 所有以value单词开头的属性（模糊查找）
    :link 选择所有未访问链接
    :visited 选择所有访问过的链接
    :active 选择活动链接
    :hover 选择鼠标在链接上时
    :focus 选择具有焦点的输入元素
    :first-letter 选择第一个字母
    :first-line 选择每一个元素的第一行
    :first-child 选择父级的第一个子集
    :before 在元素之前插入内容
    :after 在元素之后插入内容
    :lang(language) 选择一个lang属性为language的所有元素
    element1~element2 1元素之后的每一个2元素
    element[attribute^=value] 属性以value开头的元素
    element[attribute$=value] 属性以value结尾的元素
    element[attribute*=value] 属性以包含value的元素
    :first-of-type 子集第一个同类型元素
    :last-of-type 子集最后一个同类型元素
    :only-of-type 子集唯一同类型元素
    :only-child 子集唯一子元素
    :nth-child(n) 第n个子元素
    :nth-last-child(n) 倒数第n个子元素
    :nth-of-type(n) 第n个同类子元素
    :nth-last-of-type(n) 倒数第n个同类子元素
    :last-child 最后一个子元素
    :root 文档根元素
    :empty 没有子级的元素
    :target 当前活动的元素
    :enabled 已启用的输入元素
    :disabled 禁用的输入元素
    :checked 选中的输入元素
    :not(selector) 非元素
    ::selection 匹配元素中被用户选中或处于高亮状态的部分
    :out-of-range 匹配值在指定区间之外的input元素
    :in-range 匹配值在指定区间之内的input元素
    :read-write 用于匹配可读及可写的元素
    :read-only 用于匹配设置 "readonly"（只读） 属性的元素
    :optional 用于匹配可选的输入元素
    :required 用于匹配设置了 "required" 属性的元素
    :valid 用于匹配输入值为合法的元素
    :invalid 用于匹配输入值为非法的元素
  2. 自定义选择器
    3. 选择器 说明
    $("p") 元素选择器
    $("#test") #id选择器
    $(".class") .class选择器
    $("*") 选取所有元素
    $(this) 当前HTML元素
    $("p.intro") 选取 class 为 intro 的p元素
    $("p:first") 选取第一个p元素
    $("ul li:first") 选取第一个ul的第一个li元素
    $("ul li:first-child") 选取每个ul的第一个li元素
    $("[href]") 选取带有 href 属性的元素
    $("a[target='_blank']") 选取所有 target 属性值等于 "_blank" 的a元素
    $("a[target!='_blank']") 选取所有 target 属性值不等于 "_blank" 的a元素
    $(":button") 选取所有 type="button" 的input元素 和button元素
    $("tr:even") 选取偶数位置的tr元素
    $("tr:odd") 选取奇数位置的tr元素
